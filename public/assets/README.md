# مجلد الأصول (Assets)

هذا المجلد مخصص لوضع جميع الملفات الثابتة للموقع مثل:

## الصور
- `logo.png` - شعار الشركة
- `hero-image.png` - صورة رئيسية للصفحة الرئيسية
- `team/` - صور الفريق الطبي
- `services/` - صور توضيحية للخدمات
- `testimonials/` - صور العملاء
- `blog/` - صور المقالات

## الأيقونات
- `icons/` - أيقونات مخصصة
- `favicon.ico` - أيقونة الموقع

## ملفات أخرى
- `documents/` - المستندات والملفات القابلة للتحميل
- `videos/` - مقاطع الفيديو التوضيحية

## ملاحظات
- جميع الصور يجب أن تكون محسنة للويب
- استخدام تنسيقات WebP و JPEG للصور
- ضغط الصور قبل رفعها
- إضافة alt text مناسب لجميع الصور
