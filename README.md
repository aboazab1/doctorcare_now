# طبيب الرعاية - Doctor Care Website

موقع إلكتروني حديث ومتجاوب لمركز "طبيب الرعاية - Doctor Care"، وهو أحد المراكز الرائدة للخدمات الطبية المنزلية في الرياض، المملكة العربية السعودية.

## المميزات

- 🏥 **خدمات طبية منزلية شاملة** - رعاية متخصصة لجميع الفئات العمرية
- 🌐 **دعم اللغتين** - واجهة باللغة العربية والإنجليزية
- 📱 **تصميم متجاوب** - يعمل بسلاسة على جميع الأجهزة
- ⚡ **أداء سريع** - محسن للأداء والسرعة
- 🔍 **SEO محسن** - تحسين لمحركات البحث
- 🎨 **تصميم حديث** - واجهة مستخدم جميلة وسهلة الاستخدام

## التقنيات المستخدمة

- **React 18** - مكتبة JavaScript حديثة
- **Vite** - أداة بناء سريعة
- **Tailwind CSS** - إطار عمل CSS
- **Framer Motion** - مكتبة الرسوم المتحركة
- **React Router** - إدارة التنقل
- **Lucide React** - مكتبة الأيقونات
- **React i18next** - دعم اللغتين

## التثبيت والتشغيل

### المتطلبات
- Node.js 16+ 
- npm أو yarn

### خطوات التثبيت

1. **استنساخ المشروع**
   ```bash
   git clone <repository-url>
   cd doctor-care-website
   ```

2. **تثبيت التبعيات**
   ```bash
   npm install
   ```

3. **تشغيل المشروع في وضع التطوير**
   ```bash
   npm run dev
   ```

4. **بناء المشروع للإنتاج**
   ```bash
   npm run build
   ```

5. **معاينة الإنتاج**
   ```bash
   npm run preview
   ```

## الأوامر المتاحة

```bash
# تشغيل المشروع
npm run dev

# بناء المشروع
npm run build

# معاينة الإنتاج
npm run preview

# فحص الكود
npm run lint

# إصلاح أخطاء الكود
npm run lint:fix

# تحليل الحزمة

# تحسين الأداء
npm run optimize
```

├── pages/                  # صفحات الموقع
│   ├── Home.jsx            # الصفحة الرئيسية
│   ├── Services.jsx        # صفحة الخدمات
│   ├── About.jsx           # صفحة من نحن
│   ├── Contact.jsx         # صفحة الاتصال
│   ├── FAQ.jsx             # الأسئلة الشائعة

## الصفحات المتاحة

- **الرئيسية** (`/`) - الصفحة الرئيسية مع نظرة عامة على الخدمات
- **الخدمات** (`/services`) - تفاصيل جميع الخدمات الطبية
- **من نحن** (`/about`) - معلومات عن الشركة والفريق
- **اتصل بنا** (`/contact`) - معلومات الاتصال ونموذج التواصل
- **الأسئلة الشائعة** (`/faq`) - إجابات على الأسئلة الشائعة
- **المدونة** (`/blog`) - مقالات طبية ونصائح صحية
- **خريطة التغطية** (`/coverage`) - المناطق المغطاة وأوقات الاستجابة
